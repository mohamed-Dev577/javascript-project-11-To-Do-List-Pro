# 📝 To-Do List Pro

تطبيق قائمة مهام بسيط باستخدام JavaScript يتيح إضافة، حذف، وتحديد المهام كمكتملة.

---

## 🚀 فكرة المشروع

يسمح التطبيق للمستخدم بـ:
- إضافة مهمة جديدة
- حذف أي مهمة
- تحديد المهمة كمكتملة (عن طريق الضغط عليها)

---

## 🛠️ التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript (DOM)

---

## ✨ المميزات

- إضافة المهام بسهولة
- حذف المهام
- تحديد المهمة كمكتملة باستخدام `classList.toggle`
- واجهة بسيطة باللغة العربية

---

## 📂 هيكل المشروع
todo-list-pro/

---

## ▶️ طريقة التشغيل

1. افتح ملف `index.html`
2. اكتب المهمة في الحقل
3. اضغط على "إضافة"
4. اضغط على المهمة لتحديدها كمكتملة
5. استخدم زر "حذف" لإزالة المهمة
